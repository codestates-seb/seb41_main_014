## ✨ Git Commit Convention

| 태그이름 | 설명                                                  |
| -------- | ----------------------------------------------------- |
| feat     | 새로운 기능 추가                                      |
| update   | 수정 추가                                             |
| fix      | git과 관련된 버그 수정                                |
| design   | CSS 등 UI 수정                                       |
| style    | 코드 포맷 변경, 세미 콜론 누락, 코드 수정이 없는 경우 |
| refactor | 코드 리팩토링                                         |
| docs     | 문서 수정                                             |
| test     | 테스트 코드를 생성하거나 수정 하는 경우               |
